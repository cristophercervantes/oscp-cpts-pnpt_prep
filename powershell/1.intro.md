## Quick Reference Cheatsheet

### Cmdlet Structure
```
Verb-Noun  →  Get-Service   Stop-Process   New-Item
```

### Most Used Cmdlets
```powershell
Get-Service          # List system services
Get-Process          # List running processes
Get-Command          # Discover available cmdlets
Get-Help <cmdlet>    # Get help for any cmdlet
Select-Object        # Filter/select object properties
Where-Object         # Filter objects by condition
Sort-Object          # Sort output by a property
Format-Table         # Table view  (alias: ft)
Format-List          # List view   (alias: fl)
Write-Host           # Print text to screen
Clear-Host           # Clear the screen (alias: cls)
Import-Module        # Load a module
```

### Variable Cheat Sheet
```powershell
$name = "text"              # String (quotes = text)
$num  = 42                  # Integer (no quotes = number)
$arr  = "a","b","c"         # Array  → $arr[0] = "a"
$hash = @{1="Jan";2="Feb"}  # Hash table → $hash[1] = "Jan"
$global:var = "x"           # Global variable (visible outside functions)
[int]$str                   # Cast string to integer
```

### Operators Quick Reference
```powershell
-eq  -ne  -lt  -le  -gt  -ge   # Comparison
-like  -notlike  -contains      # Pattern / membership
-and  -or  -not                 # Logical
+  -  *  /  %                   # Math
=  +=  -=  *=  /=  ++  --       # Assignment
```
